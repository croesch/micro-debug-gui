NAME
	MicroDebugGui - debug micro-assembler and assembler files via a GUI

SYNOPSIS
	MicroDebugGui [-f MIC1 IJVM]
	MicroDebugGui -h|--help|-v|--version

DESCRIPTION
	Mandatory arguments to long options are mandatory for short options too.

	-f, --files MIC1 IJVM
		Debug micro-assembler-file MIC1 and assembler-file IJVM. If both files
		are valid, the debugger won't ask the user for entering file paths to
		these two files.

	-h, --help
		display this help and exit.

	-v, --version
		display version information and exit.
